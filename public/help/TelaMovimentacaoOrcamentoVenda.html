Movimentação 1 - Abrir Orçamento de Venda




Pré Requisitos:

Usuário cadastrado e logado, representações e produtos cadastrados.

       



A tabela a seguir descreve resumidamente os elementos desta janela.


Campo

Descrição

Sel. Cliente

Este campo oferece a função de opcionalmente selecionar um cliente pré cadastrado para preencher os campos de informações do cliente automaticamente.

Nome

Neste campo deve ser inserido o nome do cliente que solicitou o orçamento, salvo em caso de seleçao de um cliente pré cadastrado.

Documento (CPF ou CNPJ)

Neste campo deve ser inserido o documento do cliente que solicitou o orçamento, podendo ser CPF (pessoa física) e CNPJ (pessoa jurídica).

Telefone

Neste campo deve ser inserido o telefone do cliente que solicitou o orçamento.

Celular

Neste campo deve ser inserido o celular do cliente que solicitou o orçamento.

E-mail

Neste campo deve ser inserido o e-mail do cliente que solicitou o orçamento.

Descrição

Neste campo deve ser inserida a descrição do novo orçamento.

Vendedor

Neste campo é oferecida uma opção de opcionalmente selecionar um vendedor que possa ter intermediado a venda.

Estado de destino

Neste campo deve ser selecionado o estado para qual vai ser entregue os produtos do orçamento.

Cidade de destino

Neste campo deve ser selecionado a cidade para qual vai ser entregue os produtos do orçamento.

Peso total

Neste campo o valor do peso total vai sendo automaticamente preenchido enquanto adiciona os produros orçados.

Valor orçado dos itens

Neste campo o valor do preço orçado vai sendo preenchido automaticamente enquanto adiciona os produtos orçados.

Validade

Neste campo deve ser inserida a data da validade do novo orçamento.

Filtro

Neste campo opcionalmente pode ser inserido uma chave para procurar produtos para acição ao orçamento.

Produto seleconado

Campo preenchido automaticamente ao selecionar um produto da tabela.

Valor unitário

Neste campo o valor unitário do produto selecionado é preenchido automaticamente, mas pe permitido também alterar o valor deste.

Quantidade desejada

Neste campo deve ser inserida a quantidade desejada do produto selecionado.

Valor total

Neste campo o valor total é preenchido automaticamente conforme é inserido a quantidade.



Ações: <Descrição dos comandos de interação>

Botão                        Descrição

Limpar (item)        Limpa os itens do orçamento e os valores do peso total e valor orçado.

Adicionar (item)        Habilita o subformulário para inserir um item ao orçamento.

Cancelar                Limpa todos os campos do formulário, os itens e retorna para a página de controle de orçamentos.

Limpar                        Limpa todos os campos do formulário e os itens.

Salvar                        Valida os campos preenchidos, formata valores em formato diferente e manda os dados para o servidor processar e                                        gravar no banco de dados.

Filtrar                        Filtra os produtos cadastrados pela chave inserida no campo filtro.

Cancelar (item)        Limpa os dados de algum produto selecionado e fecha o subformulário de adição de produtos ao orçamento.

Continuar                Valida os dados inseridos sobre o produto selecionado, atualiza a tabela de itens e os valores de peso e valor orçado.



Restrições:

- Não é permitido a inserção de cpf ou cnpj inválido ou com tamanho inválido.

- Não é permitido a inserção de telefone ou celular ou e-mail do cliente com formato inválidos.

- Não é permitido a adição de itens ao orçamento sem selecionar o estado de destino previamente.

- Não é permitida a abertura de um oçamento sem os produtos serem previamente cadastrados.

- Não é permitido o salvamento da abertura do orçamento de venda sem adicionar produtos.
