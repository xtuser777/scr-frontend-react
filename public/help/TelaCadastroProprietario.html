Cadastro Básico 4 - Proprietários

 




                 Explicação dos campos:

                   Motorista: opção de escolher um motorista já cadastrado para preencher o restante dos campos.

Tipo: define o tipo de pessoa do proprietário: PESSOA FÍSICA, PESSOA JURÍDICA.

Nome, Nascimento, Rg, Cpf (Tipo: PESSOA FÍSICA): informações pessoais de cada proprietário, do tipo física, data de nascimento no                        formato DD/MM/AAAA, cpf no formato ###.###.###-##.

Razão Social, Nome Fantasia, Cnpj (Tipo: PESSOA JURÍDICA): informações pessoais de cada proprietário, do tipo jurídica, cnpj no formato                ##.###.###/####-##.

Rua, Número, Bairro, Complemento, Estado, Cidade, Cep: dados do endereço do proprietário, cep no formato ##.###-###.

Telefone, Celular, E-mail: dados de contato do proprietário, telefone no formato (##) ####-####, celular no formato (##) #####-####, e-mail                no formato usuário@domínio.

                   Campos obrigatórios: Nome (caso tipo pessoa física), Nascimento (caso tipo pessoa física), Rg (caso tipo pessoa física), Cpf (caso tipo pessoa                        física), Razão Social (caso tipo pessoa jurídica), Nome Fantasia (caso tipo pessoa jurídica), Cnpj (caso tipo pessoa jurídica), Rua, Número,                        Bairro, Estado, Cidade, Cep, Telefone, Celular, E-mail.
