Movimentação 2 - Abrir Pedido de Venda




Pré Requisitos:

Usuário cadastrado e logado, formas de  pagamento, clientes, representações e produtos cadastrados.

       



A tabela a seguir descreve resumidamente os elementos desta janela.


Campo

Descrição

Orçamento

Neste campo é opcionalmente oferecida um opção de selecionar um orçamento de venda aberto para carregar o restante dos dados que estão presentes no orçamento automaticamente.

Descrição

Neste campo deve ser inserida a descrição do novo pedido.

Cliente

Neste campo deve ser selecionado o cliente que solicitou o pedido (pode vir de um orçamento).

Estado de destino

Neste campo deve ser selecionado o estado para qual vai ser entregue os produtos do pedido.

Cidade de destino

Neste campo deve ser selecionado a cidade para qual vai ser entregue os produtos do pedido.

Vendedor

Neste campo é oferecida uma opção de opcionalmente selecionar um vendedor que possa ter intermediado a venda.

Porcentagem da comissão ao vendedor

Neste campo deve ser inserida a porcentagem de comissão paga ao vendedor (se um vendedor estiver selecionado).

Peso total

Neste campo o valor do peso total vai sendo automaticamente preenchido enquanto adiciona os produros.

Valor dos itens

Neste campo o valor do preço vai sendo preenchido automaticamente enquanto adiciona os produtos.

Forma de pagamento

Neste campo deve ser seleciona a forma de pagamento utilizada pelo cliente.

Filtro

Neste campo opcionalmente pode ser inserido uma chave para procurar produtos para acição ao orçamento.

Produto seleconado

Campo preenchido automaticamente ao selecionar um produto da tabela.

Valor unitário

Neste campo o valor unitário do produto selecionado é preenchido automaticamente, mas pe permitido também alterar o valor deste.

Quantidade desejada

Neste campo deve ser inserida a quantidade desejada do produto selecionado.

Valor total

Neste campo o valor total é preenchido automaticamente conforme é inserido a quantidade.



Ações: <Descrição dos comandos de interação>

Botão                        Descrição

Limpar (item)        Limpa os itens do pedido e os valores do peso total e valor total.

Adicionar (item)        Habilita o subformulário para inserir um item ao pedido.

Cancelar                Limpa todos os campos do formulário, os itens e retorna para a página de controle de pedidos.

Limpar                        Limpa todos os campos do formulário e os itens.

Salvar                        Valida os campos preenchidos, formata valores em formato diferente e manda os dados para o servidor processar e                                        gravar no banco de dados.

Filtrar                        Filtra os produtos cadastrados pela chave inserida no campo filtro.

Cancelar (item)        Limpa os dados de algum produto selecionado e fecha o subformulário de adição de produtos ao pedido.

Continuar                Valida os dados inseridos sobre o produto selecionado, atualiza a tabela de itens, de comissões das representações e                                os valores de peso e valor orçado.



Restrições:

- Não é permitido a adição de itens ao pedido sem selecionar o estado de destino previamente.

- Não é permitida a abertura de um pedido sem os produtos e os clientes serem previamente cadastrados.

- Não é permitido o salvamento da abertura do pedido de venda sem adicionar produtos.
