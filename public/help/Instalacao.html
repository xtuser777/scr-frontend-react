Instalando e Configurando o Banco de Dados

O SGBD MariaDB já vem instalado no servidor de hospedagem, sendo necessário somente a configuração do mesmo com os passos a seguir:

1 - No painel de controle do servidor (cPanel), vá até a sessão BANCO DE DADOS e entre em Banco de dados MYSQL(R);

2 - Na sessão Criar novo banco de dados, informe o nome do novo banco de dados a ser usado pelo sistema, no caso scr, que no final ficará globotransportes_scr, depois clique em Criar banco de dados;

3 - Na sessão Usuários do MySql, informe o nome do usuário que terá acesso ao novo bando de dados criado, no caso também scr, que no final ficará gobotransportes_scr, informe uma senha segura para o usuário informado para proteger esta conta, o banco a ser vinculado e o servidor, depois clique em Criar usuário;

4 - Na sessão Adicionar usuário ao banco de dados, selecione o usuário que você criou, no caso, globotransportes_scr e depois o banco de dados criado para o sistema, no caso também globotransportes_scr e clique em Adicionar. Na área de privilégios, marque a opção TODOS OS PRIVILÁGIOS e clique em Fazer alterações.

5 - No painel do servidor novamente, na sessão BANCO DE DADOS, entre em phpmyadmin, no painel lateral, selecione o banco de dados globotransportes_scr e depois vá até a aba importar, enconte o arquivo scrmysql.sql no cd e importe como SQL.


 Instalando e Configurando o Software

Copie a pasta do sistema que está no CD para a área de trabalho e abra uma conexão FTP com o servidor com o cliente CoreFTP com o usuário globotransportes, servidor ftp.globotransportes.com.br e senha de acesso do painel do servidor. Depois na pasta do servidor, entre em public_html e peque a pasta do sistema na área de trabalho e cole para este diretório no servidor.

Qualquer dúvida ou problemas, acione o supote imediatamente para iniciar um acesso remoto.


 Primeiro Acesso

    Um membro do suporte ou desenvolvimento, presencialmente ou por acesso remoto fará login com a conta suporte e o administrador da empresa fará o cadastro da parametrização e de seu usuário e senha, depois o membro do suporte conferirá os dados cadastrados e sairá do sistema para o administrador fazer seu login e começar seus cadastros.
